## Story

**As a** network engineer,
**I want** to view LLDP/CDP neighbor data from devices,
**so that** I can map physical topology.

---

## Acceptance Criteria

1. `discover_lldp_cdp` returns LLDP/CDP neighbor data per device port
2. Data includes neighbor device name, port, platform, and IP
3. Integration into `full_discovery()` output
4. Natural language: "show me the network topology" or "who is connected to switch X?" works
