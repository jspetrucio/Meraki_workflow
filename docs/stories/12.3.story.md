## Story

**As a** network admin,
**I want** to configure switch port schedules,
**so that** ports are enabled/disabled at specific times through natural language.

---

## Acceptance Criteria

1. Full CRUD for port schedules with time windows (create, update, delete)
2. `discover_port_schedules` returns all schedules with assigned ports
3. Assignable to specific ports via switch port configuration
4. Natural language: "disable lab ports after 6pm" works
5. Backup created before any schedule change
6. Dry-run mode supported: preview schedule changes before applying
7. Port schedule status integrated into `full_discovery()` output
