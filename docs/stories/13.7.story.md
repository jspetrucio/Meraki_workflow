## Story

**As a** network admin,
**I want** to configure NetFlow export settings,
**so that** I can send flow data to a collector.

---

## Acceptance Criteria

1. `discover_netflow_config` returns current NetFlow collector IP, port, and reporting status
2. `configure_netflow` enables/disables NetFlow and sets collector destination
3. Backup created before any NetFlow config change
4. Backup + rollback supported
5. Integration into `full_discovery()` output
6. Natural language: "enable NetFlow to collector 10.1.1.100 port 2055" works
